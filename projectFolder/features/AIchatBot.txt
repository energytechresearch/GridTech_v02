## **Functional Requirements (Excerpt)**

**FR-1: Natural Language Query**

* User enters a question
* System translates to structured DB query or retrieval pipeline
* AI composes response using contextual rules

**FR-2: Result Transparency**

* Responses include:

  * Linked records
  * Dataset scope
  * Timestamp

**FR-3: Guardrails**

* AI cannot:

  * Modify records
  * Approve workflows
  * Change lifecycle states

* Text explanation
* Tabular summaries
* KPI rollups
* Suggested follow-up prompts
